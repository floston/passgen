# Password Generator (passgen)

Консольное приложение для генерации криптографически стойких паролей.

## Особенности

- **Криптографически стойкий ГСЧ** — использует `BCryptGenRandom` (Windows) или `/dev/urandom` (Linux)
- **Равномерное распределение** — rejection sampling для устранения модулярной предвзятости
- **Кроссплатформенность** — работает на Windows и Linux
- **Безопасная очистка памяти** — пароль удаляется из памяти после использования
- **Три уровня сложности** — simple, medium, complex
- **Генерация нескольких паролей** — одной командой
- **Вывод в файл** — опционально

## Сборка

### Windows (MinGW)

```bash
gcc -o passgen.exe passgen.c -lbcrypt
```

### Linux

```bash
gcc -o passgen passgen.c
```

### Используя Makefile

```bash
# Сборка
make

# Очистка
make clean

# Пересборка
make rebuild

# Тесты
make test
```

## Использование

```
passgen [ключи]
```

### Ключи

| Ключ | Описание | По умолчанию |
|------|----------|--------------|
| `-l, --length N` | Длина пароля (6-128) | 12 |
| `-t, --type TYPE` | Тип сложности: `simple`, `medium`, `complex` | medium |
| `-n, --count N` | Количество паролей (1-1000) | 1 |
| `-o, --output FILE` | Вывод в файл | stdout |
| `-h, --help` | Показать справку | — |

### Примеры

```bash
# Пароль средней сложности длиной 12 (по умолчанию)
passgen

# Сложный пароль длиной 20
passgen -l 20 -t complex

# 5 паролей средней сложности
passgen -n 5

# 10 паролей в файл
passgen -n 10 -o passwords.txt

# Простой пароль длиной 8
passgen -l 8 -t simple
```

## Требования к паролям

| Тип | Символы | Требования |
|-----|---------|------------|
| **simple** | Строчные латинские буквы (a-z) | Только буквы нижнего регистра |
| **medium** | Строчные + заглавные буквы + цифры | Минимум 1 заглавная буква и 1 цифра |
| **complex** | Все выше + спецсимволы (`@$%&*!#?`) | Минимум 1 символ из каждого набора |

## Наборы символов

- **Строчные**: `abcdefghijkmnopqrstuvwxyz` (без `l` и `o` для читаемости)
- **Заглавные**: `ABCDEFGHJKLMNPQRSTUVWXYZ` (без `I` и `O` для читаемости)
- **Цифры**: `23456789` (без `0` и `1` для читаемости)
- **Спецсимволы**: `@$%&*!#?`

## Безопасность

1. **Криптографический ГСЧ**: Использует системный генератор случайных чисел (CSPRNG)
2. **Равномерное распределение**: Rejection sampling исключает модулярную предвзятость
3. **Очистка памяти**: `secure_zero()` предотвращает утечку пароля в памяти
4. **Предсказуемые символы**: Исключены похожие символы (`0`/`O`, `1`/`l`/`I`)
